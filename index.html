Hydroid Website

Hydroid: An IoT-Based Hydroponic Nutrient Film Technique for Lettuce Farming
This project was made to comply with school's academic requirements. Hydroid is an IoT-Based Hydroponics system that has a system for monitoring and controlling the hydroponics system.
